# 📰 Fake News Detection System

## 🚀 Overview

This project detects whether a news headline or article is **Real or Fake** using Machine Learning and BERT.

---

## ✨ Features

* Detect fake/real news
* Analyze live news
* Shows confidence score
* Shows “Uncertain” if models disagree

---

## ⚙️ How to Run

1. Install libraries:
   pip install streamlit scikit-learn pandas numpy transformers torch requests

2. Add API key in `api.py`

3. Run:
   streamlit run app.py

---

## 🧪 Usage

* Enter news → Click Predict
* Or fetch live news

---

## ⚠️ Note

This model detects patterns, not real-world facts.
